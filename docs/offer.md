# Business Website Offer

## Starter Website

A simple, polished website for a local business that needs to look credible, load quickly, and turn visitors into calls or quote requests.

Includes:

- 4-5 pages
- Mobile-friendly design
- Contact form
- Clear phone CTA
- Services section
- About / trust section
- Gallery or project examples
- Testimonials or reviews section
- FAQ section
- Basic local SEO
- Static site launch

## Pricing Tiers

### Starter

Best for a business with weak online presence that needs a clean website quickly.

- 4-5 pages
- One selected template
- Client-provided photos or tasteful placeholders
- Basic SEO title and description
- Contact form setup
- Static deployment

### Professional

Best for businesses that want more trust-building and conversion features.

- Everything in Starter
- Expanded services page
- Process section
- FAQ improvements
- Service-area copy
- More complete gallery / proof section
- Package or consultation section if relevant

### Monthly Care

Best for clients who want the site kept fresh after launch.

- Small copy updates
- Photo swaps
- Testimonial additions
- Seasonal CTA updates
- Basic uptime and form checks
- Light SEO/content recommendations

## Positioning

This is not a custom app, CRM, dashboard, or marketing automation system. It is a fast, professional static website package for small local businesses that need a better first impression and clearer customer action.
